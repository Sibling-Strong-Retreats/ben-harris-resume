# Ben Harris - Resume

A modern, elegant resume website built with React and Vite.

## Development

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```

## Deployment

This site automatically deploys to GitHub Pages when you push to the `main` branch.

The site will be available at: `https://[your-username].github.io/ben-harris-resume/`

## Tech Stack

- React 18
- Vite
- GitHub Pages
- GitHub Actions
